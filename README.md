# Image-Steganography
A LSB and GAN based Image Steganography application.

# Working

1. Run the python file.
2. Select an Image you want to hide the message into or get the message from.
3. Choose your Method to hide the message or the method throught which the message was hidden.
4. Type the message and Click on hide Message or extract message accordingly.


* Use python 3.7 since no support for SteganoGAN module in python 3.7+
